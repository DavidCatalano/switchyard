### Usage Notes
- Each research agent writes to its own CONTEXT file using this template. Include the sections relevant to that agent; remove or leave stub sections that don't apply.
- Use descriptive slugs to differentiate CONTEXT files within the same SEP (e.g., `-CONTEXT-mux-mvp` for Librarian, `-CONTEXT-mux-external-research` for Surveyor).
- Multiple CONTEXT documents per SEP are expected. The doc index tells the creation order.
